# 19. Conflict of Interest

The authors declare that they are contributors to the Decision Security Engineering framework and the associated PrivateVault reference implementation described in this paper.

The engineering framework itself is intended to remain implementation independent, and alternative implementations may realize the same architectural principles using different technologies and design choices.

No external organization influenced the technical content or conclusions presented in this paper.
